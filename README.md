# Brewery List App

A React app that fetches and displays breweries with pagination and search using the [Open Brewery DB API](https://www.openbrewerydb.org/).

## Features
- Fetch brewery data from API
- Search breweries by name
- Pagination (Previous/Next page)
- Responsive table for brewery details
- Error handling for API requests

## Tech Stack
- React (Vite)
- Axios (API requests)
- Tailwind CSS (styling)
